# Mithran MV — Full Stack · Cloud Systems · AI & Blockchain

Portfolio showcasing production systems, ETHGlobal wins, and internship projects across AI, Web3, enterprise, and IoT.

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Static data-driven content (no backend)

## Routes
- `/` — Home overview
- `/experience` — Timeline, skills, operating principles
- `/projects` — Individual projects (filterable)
- `/projects/[slug]` — Case studies
- `/internship` — Internship projects
- `/awards` — ETHGlobal + certifications
- `/contact` — Contact form and links

npm install
npm run dev      # http://localhost:3000
